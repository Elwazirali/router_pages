AwkwardTurtle

An awesome Chatting application made using web sockets and facial recognition software!
Detects your facial expression while you chat with your friends, and shows you a report of how awkward the interaction was based on your face and nature of conversation.

ChromicSans

A Chrome extension built using Google's voice recognition API to browse the internet. Designed to provide a completely hands-free experience. Currently does things like, open a new tab, go to site, search on Google or Youtube, and much more.

GymFlow

This is a phone and web social network platform made using ionic, Angular, and Firebase. The idea is to create a social network for athletes and people participating in any kind of gym or sports group. This application will enable users to post questions, answer questions, recognize one another, and post their athletic achievements. The application is currently able to onboard users so feel free to try it out and post.

Trendify

 the code

A sentiment analysis web application made using Node for the backend and Angular for the front end. Essentially type in a topic, item, or anything to get an overview of the general sentiment towards that topic. This application uses a machine learning API to process the topic data from Twitter and The New York Times.
